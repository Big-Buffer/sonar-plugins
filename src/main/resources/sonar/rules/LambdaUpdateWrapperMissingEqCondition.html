此规则检查 MyBatis Plus 的 LambdaUpdateWrapper 是否包含至少一个 eq 条件。 如果没有 eq 条件，可能会导致全表更新，这是非常危险的操作。

不合规代码示例

LambdaUpdateWrapper<User> updateWrapper = Wrappers.lambdaUpdate();
updateWrapper.set(User::getName, "newName");
userMapper.update(null, updateWrapper); // 缺少 eq 条件

合规代码示例

LambdaUpdateWrapper<User> updateWrapper = Wrappers.lambdaUpdate();
updateWrapper.eq(User::getId, 1L) // 包含 eq 条件
             .set(User::getName, "newName");
userMapper.update(null, updateWrapper);
